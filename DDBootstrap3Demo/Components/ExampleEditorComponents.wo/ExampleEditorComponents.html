With this set of Editor-Components you can build nice looking Editors without much Mark-Up. Quick and rapid development.

Wrapps your content into logical pieces.

In order for proper responsive layouting we need BSEditorForm.

<wo:BSEditorForm>Editor-Components here</wo:BSEditorForm>

The plain BSEditorRow is for custom content. It can have a label which is totally optional.

This is an Editor row

This is an Editor row with a label!

 <wo:BSEditorRow><p>This is an Editor row</p></wo:BSEditorRow> <wo:BSEditorRow label="My Label"><p>This is an Editor row with a label!</p></wo:BSEditorRow>

You know BSBrowser from the Component Example? Well here is the Editor Row version of it!

<wo:BSEditorRowBrowser list="$exampleList" selections="$exampleSelections" label="Available Options" multiple="$true" />

The well known BSCheckBox as a BSEditorRow.

<wo:BSEditorRowCheckBox label="Online" checked="$exampleSwitch" />

Always wanted an easy way to select a color? Don't look further here is your solution!
You can also give the User a hint with the helpText binding!

<wo:BSEditorRowColorPicker label="Background Color" value="$exampleDict.color" helpText="Format: #RRGGBB" />

We've support for simple PopUpButtons as well! You can also add some custom content which gets displayed after the PopUpButton.

m/s^2 <wo:BSEditorRowPopUpButton label="List 1" list="$exampleList" selection="$exampleDict.selection" noSelectionString="- Select an Entry -" /> <wo:BSEditorRowPopUpButton label="List 2" list="$exampleList" selection="$exampleDict.selection" noSelectionString="- Select an Entry -">m/s^2</wo:BSEditorRowPopUpButton>

Shortcut for BSBrowser, multiple is forced to true in this Component.

<wo:BSEditorRowSelectMultiple label="Many Items" list="$exampleList" selections="$exampleSelections" />

All those manual input fields for an address. Don't panic any more, we've only 2 components which'll do the work for you!
BTW: Placeholder support is on board!

<wo:BSEditorRowStreetAndNo label="Address" street="$exampleDict.street" streetPlaceholder="Main Street" houseNo="$exampleDict.houseNo" houseNoPlaceholder="42" /> <wo:BSEditorRowZIPCityDistrict label="City" city="$exampleDict.city" cityPlaceholder="Durham" zip="$exampleDict.city" zipPlaceholder="75226" district="$exampleDict.district" districtPlaceholder="No 9" />

Textarea support in 4 sizes!

<wo:BSEditorRowTextarea label="Normal" value="$exampleDict.textarea" /> <wo:BSEditorRowTextareaMedium label="Medium" value="$exampleDict.textarea" /> <wo:BSEditorRowTextareaLarge label="Large" value="$exampleDict.textarea" /> <wo:BSEditorRowTextareaXtraLarge label="XtraLarge" value="$exampleDict.textarea" />

Last but not least: TextField in all it's glory!

<wo:BSEditorRowTextField label="Some Text" value="$exampleDict.textfield" helpText="Just enter something here" required="$true" unit="text/s" />
